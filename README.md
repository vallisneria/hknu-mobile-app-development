# 모바일앱개발
한경대학교 3학년

## 교과목개요 및 학습목표
[비대면 수업]
이 과목에서는 JavaScript 풀 스택(MongoDB, Express, AngularJS, Node.js)을 이용한 웹 애플리케이션 개발 방법을 공부하며, 특히 Bootstrap을 적용하여 Mobile-ready 웹 앱을 개발하고 클라우드 플랫폼 서비스에 배포하는 과정을 프로젝트 기반으로 학습한다.
